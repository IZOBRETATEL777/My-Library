Краткое описание

Программа «Моя библиотека» создана для облегчения контроля за книгами, находящихся в Вашей домашней библиотеке. Данная программа ведет учет ваших книг и распределяет их по нескольким категориям:

    • Название книги

    • Автор

    • Жанр - жанр, которому относится книга

    • Серия - из какой серии Ваша книга

    • Год - год написания

    • Есть в наличии - есть ли эта книга в Вашей библиотеке, либо Вы ее одолжили/обменяли

Показать список книг

Для того, чтобы просмотреть текущий список книг в базе данных программы, кликните по пункту «Показать список книг». Вам будет представлена таблица, в которой отражена ваша текущая библиотека (те книги, что вы добавили посредством данной программы в пункте меню «Добавить книгу», см. следующий пункт). Кликнув по кнопке «Редактировать» и дважды кликнув по ячейке таблицы вы можете изменить значение этой ячейки. Правила заполнения ячеек отмечены в следующем пункте. Для того, что бы сохранить изменения кликните по «Сохранить изменения» и согласитесь с кликнув по кнопке «Да». Для отмены выберите «Нет», либо кликните по кнопке «Отменить изменения».

Добавить книгу

Для того, чтобы добавить новую книгу в базу данных кликните по кнопке «Добавить книгу» в главном меню программы. Вам откроется форма которую вы должны заполнить. Значения некоторых полей в форме описаны в пункте «Описание». Заполнение таких полей как «Название книги» и «Автор» являются обязательными. Без них введенная Вами информация не будет добавлена в базу данных программы. Также в поле «Год» введеный текст должен, желательно, представлять набор целых арабских цифр. Если введенная информация будет удовлетворять вышесказанным правилам, она будет добавлена базу данных программы.

Дополнительные функции программы «Моя библиотека» (Главное меню программы  →  Дополнительно)

  • Сохранить базу в .xlsx документ - сохраняет базу данных библиотеки в файл в формате .xlsx для дальнейшей работы в редакторе таблиц. Кликните по данному пункту всплывающего меню и выберите путь для сохранения. После нажатия кнопки «Сохранить» файл сохранится как Library#.xlsx, где # - номер файла.
  • Очистить базу данных - безвозвратно очистить текущую базу данных.
